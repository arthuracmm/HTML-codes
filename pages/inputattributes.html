HTML Input Attributes

Este capítulo descreve os diferentes atributos do elemento HTML<input>.

O atributo value

O atributo input value especifica um valor inicial para um campo de entrada:

                
                    <form>
                        <label for="fname"> Primeiro Nome </label><br>
                        <input type="text" id="fname" name="flname" value="John"><br>

                        <label for="fname"> Sobrenome </label><br>
                        <input type="text" id="lname" name="lname" value="doe"><br><br>

                    </form

O atributo Readonly

O atributo input readonly especifica que um campo de entrada é somente leitura.

                
                    <form>
                        <label for="fname"> Primeiro Nome </label><br>
                        <input type="text" id="fname" name="flname" value="John" readonly><br>

                        <label for="fname"> Sobrenome </label><br>
                        <input type="text" id="lname" name="lname" value="doe"><br><br>

                    </form

O atributo Disable

O atributo input disable especifica que um campo de entrada deve ser desabilitado.

                
                    <form>
                        <label for="fname"> Primeiro Nome </label><br>
                        <input type="text" id="fname" name="flname" value="John" disable><br>

                        <label for="fname"> Sobrenome </label><br>
                        <input type="text" id="lname" name="lname" value="doe"><br><br>

                    </form
