HTML Styles

O atributo de estilo HTML é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Background Color

A propriedade CSS background-color define a cor de fundo de um elemento HTML.

                        
                            <body style = "background-color: #400000;">
                            
                            <h1>Isso é um cabeçalho</h1>
                            <p>Isso é um paragrafo</p>
                            
                            </body>

Resultado

Isso é um cabeçalho.

Isso é um paragrafo.
