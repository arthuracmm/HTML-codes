HTML Form Attributes

Este capítulo descreve os diferentes atributos do elemento HTML <form>.

O atributo de ação

O atributo action define a ação a ser executada quando o formulário for enviado.

Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.

In the example below, the form data is sent to a file called "action_page.php". This file contains a server-side script that handles the form data:

                
                    <form action="/action_page.php">
                        <label for="fname"> Primeiro Nome </label><br>
                        <input type="text" id="fname" name="flname" value="doe"><br>

                        <label for="fname"> Sobrenome </label><br>
                        <input type="text" id="fname" name="flname" value="doe"><br><br>

                        <input type="submit" value="submit">

                    </form

O atributo target

O atributo target especifica onde exibir a resposta recebida após o envio do formulário.

                
                    <form action="/action_page.php" target="_blank">

O Atributo metod

O atributo method especifica o método HTTP a ser usado ao enviar os dados do formulário.

Os dados do formulário podem ser enviados como variáveis ​​​​de URL (com metod = "get") ou como pós-transação HTTP (com metod = "post").

                
                    <form action="/action_page.php" method="get">

Metodo Post

                
                    <form action="/action_page.php" method="post">
