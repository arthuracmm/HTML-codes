HTML Formatting

HTML contém vários elementos para definir texto com um significado especial.

HTML Formatting Elements

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

<b>Texto em negrito

<strong>Texto importante

<i>Texto em itálico

<em>Texto enfatizado

<mark>Texto marcado

<small>Texto menor

<del>Texto excluído

<ins>Texto inserido

<sub>Texto subscrito

<sup>Texto sobrescrito

                        
                            <b>Esse texto é negrito </b>

                            <strong>Esse texto é importante</strong>

                            <i>Esse texto é itálico</i>

                            <em>Esse texto está enfatizado</em>

                            <small>Este é um texto menor</small>

Resultado

Esse texto é negrito Esse texto é importante Esse texto é itálico Esse texto está enfatizado Este é um texto menor.
