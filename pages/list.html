HTML Lists

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Unordered HTML List

Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>.

Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

                
                        <ul>
                            <li>Cafe</li>
                            <li>Cha</li>
                            <li>Leite</li>
                        </ul>

Resultado

  • Cafe
  • Cha
  • Leite

Ordered HTML List

Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

                
                        <ol>
                            <li>Cafe</li>
                            <li>Cha</li>
                            <li>Leite</li>
                        </ol>

Resultado

    1. Cafe
    2. Cha
    3. Leite
