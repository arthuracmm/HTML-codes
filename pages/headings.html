HTML Headings

O que são atributos HTML?

Os títulos HTML são títulos ou legendas que você deseja exibir em uma página da web.

Os títulos HTML são definidos com as tags <h1> a <h6>.

<h1> define o título mais importante.

<h6> define o título mais importante.

Estrutura de um atributo

                        
                            <h1>Heading 1</h1>
                            <h2>Heading 2</h2>
                            <h3>Heading 3</h3>
                            <h4>Heading 4</h4>
                            <h5>Heading 5</h5>
                            <h6>Heading 6</h6>
