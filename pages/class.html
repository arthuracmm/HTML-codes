HTML class Attribute

O atributo<class> HTML é usado para especificar uma classe para um elemento HTML.

Vários elementos HTML podem compartilhar a mesma classe.

Usando o atributo class

O atributo <class> é frequentemente usado para apontar para um nome de classe em uma folha de estilo. Também pode ser usado por JavaScript para acessar e manipular elementos com o nome de classe específico.

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                    <style>
                        .note {
                            font-size: 120%;
                            color: red;
                        }
                    </style>
                    </head>
                    <body>

                    <h1>Meu Cabeçalho e muito<span class = note> Importante! </span> </h1>

                    <p>Esse texto e muito <span class = note> Importante! </span> </p>

                    </body>
                    </html>

Resultado

Meu Cabeçalho e muito Importante!

Esse texto e muito Importante!
